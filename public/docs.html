üìö Shoe API Documentation

API untuk mengakses database sepatu terlengkap dengan berbagai brand dan kategori.

Base URL: https://api.shoeapi.com/api/v1
Version: 1.0.0
Format: JSON

Fitur Utama

  • ‚úÖ RESTful API dengan response JSON
  • ‚úÖ Autentikasi menggunakan API Key
  • ‚úÖ Rate limiting yang fleksibel
  • ‚úÖ Pagination untuk dataset besar
  • ‚úÖ Filter dan search yang powerful

üîê Authentication

Semua request memerlukan API Key yang valid di header.

Mendapatkan API Key

  1. Daftar akun di halaman registrasi
  2. Login ke dashboard Anda
  3. Generate API Key baru
  4. Simpan API Key dengan aman

Cara Menggunakan

Sertakan API Key di header X-API-Key:

HTTP Header 
X-API-Key: your_api_key_here
‚ö†Ô∏è Penting: Jangan bagikan API Key atau simpan di repository publik.

JavaScript

üëü Get All Shoes

Mendapatkan daftar semua sepatu dengan pagination.

GET /shoes

Query Parameters

Parameter Type Required Description
page integer No Nomor halaman (default: 1)
limit integer No Items per page (default: 10, max: 100)
category string No Filter by category
brand string No Filter by brand

Request Example

cURL 
curl -X GET "https://api.shoeapi.com/api/v1/shoes?page=1&limit=10" \
  -H "X-API-Key: your_api_key_here"

Response Example

200 OK 
{

üîé Search Shoes

Mencari sepatu berdasarkan keyword.

GET /shoes/search

Query Parameters

Parameter Type Required Description
q string Yes Keyword pencarian
page integer No Nomor halaman

Request Example

Python 
import requests

response = requests.get(
    'https://api.shoeapi.com/api/v1/shoes/search',
    headers={'X-API-Key': 'your_api_key_here'},
    params={'q': 'nike', 'page': 1}
)
print(response.json())

‚ùå Error Codes

API menggunakan HTTP status codes standar.

Code Meaning Description
200 OK Request berhasil
400 Bad Request Parameter tidak valid
401 Unauthorized API Key tidak valid
404 Not Found Resource tidak ditemukan
429 Too Many Requests Rate limit exceeded
500 Server Error Internal server error

Error Response Format

{
    "success": false,
    "message": "API Key tidak valid",
    "error": {
        "code": "INVALID_API_KEY",
        "details": "The API key provided is invalid"
    }
}
