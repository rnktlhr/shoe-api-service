üìö API Documentation

API untuk mengakses database sepatu terlengkap dengan berbagai brand dan kategori

Base URL: https://api.shoeapi.com/api/v1
Version: 1.0.0
Format: JSON

Fitur Utama

  • RESTful API dengan response JSON
  • Autentikasi menggunakan API Key
  • Rate limiting yang fleksibel
  • Pagination untuk dataset besar
  • Filter dan search yang powerful

üîê Authentication

Semua request memerlukan API Key yang valid di header.

Mendapatkan API Key

1

Daftar akun di halaman registrasi

2

Login ke dashboard Anda

3

Generate API Key baru

4

Simpan API Key dengan aman

Cara Menggunakan

Sertakan API Key di header X-API-Key:

HTTP Header 
X-API-Key: your_api_key_here

Penting!

Jangan bagikan API Key atau simpan di repository publik.

‚ö° Quick Start

request.js

üëü Get All Shoes

Mendapatkan daftar semua sepatu dengan pagination.

GET /shoes

Query Parameters

Parameter Type Required Description
page integer No Nomor halaman (default: 1)
limit integer No Items per page (max: 100)

Request Example

cURL 
curl -X GET "https://api.shoeapi.com/api/v1/shoes?page=1&limit=10" \
  -H "X-API-Key: your_api_key_here"

Response Example

200 OK 
{
  "success": true,
  "data": [
    {
      "id": 1,
      "brand": "Nike",
      "model": "Air Max 90",
      "category": "Running",
      "price": 1299000,
      "stock": 15
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 10,
    "total": 150
  }
}

üîé Search Shoes

Mencari sepatu berdasarkan keyword.

GET /shoes/search
Python 
import requests

response = requests.get(
    'https://api.shoeapi.com/api/v1/shoes/search',
    headers={'X-API-Key': 'your_api_key_here'},
    params={'q': 'nike', 'page': 1}
)
print(response.json())
