üìö Quick Start Guide

Gunakan API Key Anda untuk mengakses Shoe API:

1. Get All Shoes

GET https://api.shoeapi.com/api/v1/shoes
Headers: {
    "X-API-Key": "your_api_key_here"
}

2. Get Shoe by ID

GET https://api.shoeapi.com/api/v1/shoes/{id}
Headers: {
    "X-API-Key": "your_api_key_here"
}

3. Search Shoes

GET https://api.shoeapi.com/api/v1/shoes/search?q=nike
Headers: {
    "X-API-Key": "your_api_key_here"
}

4. Filter by Category

GET https://api.shoeapi.com/api/v1/shoes?category=Running
Headers: {
    "X-API-Key": "your_api_key_here"
}
